# 센텐스크래프트 (Sentence Craft) — 베타

글로 싸우는 전쟁 — 커뮤니티 베타 뼈대.

## 빠른 시작

```bash
npm install
npm start
```

브라우저: `http://localhost:3000`  
(Supabase 쓰려면 `.env.example` 을 `.env` 로 복사 후 URL·키 입력)
