# `config` 폴더 — 쉽게 말하면

**“게임 규칙 표”**만 모아 둔 곳입니다. 코드가 길어지면 헷갈리니까, **영토**와 **정치색**을 나눴어요.

| 파일 | 안에 든 것 |
|------|------------|
| `world-territories.js` | 보수·중도·진보 3단, 게시판 4단계 설명, 인구 구간별 “마을 발전” 예시 |
| `kantapbiya.js` | **깐따삐아만** — 좌/우 영토 + 유배·귀환 규칙(예전에 나뉘어 있던 것을 한 이름으로 묶음) |
| `alignment-system.js` | 글/댓글/좋아요 같은 **점수 종류 이름**, 자동 편입 **숫자 뼈대**, AI 주제 채널 이름 |
| `signup-countries.js` | 가입 시 고를 **나라 목록**(약 250개국, ISO 2글자 + 한글·영문 이름) |

숫자나 문구를 바꿀 때는 보통 **이 폴더만** 고치면 됩니다.
