# 센텐스크래프트 — 사장님이 말씀하신 기획 정리

**한 줄 요약:**  
다양한 정치 성향의 사람들이 **게시판**으로 사회 이슈를 말하고, **글·댓글·반응**으로 정치색이 쌓이면 **어느 진영 영토**로 자동 편입되는 **글로 싸우는 전쟁** 커뮤니티입니다.

---

## 1) 세상 지도(영토) — 그림으로

```
        [ 보수 진영 ]     [ 중도·일반 ]     [ 진보 진영 ]
              |                  |                  |
              +------------------+------------------+
                                 |
                    별도 행성 [ 깐따삐아 ]
                    /              \
            [ 극좌 영토 ]      [ 극우 영토 ]
```

- 가운데 **중도**는 “연습장 / 중립”에 가깝게 둡니다.  
- **깐따삐아**는 극단 성향이 모이는 **별도 공간**이고, 그 안에서 **좌·우**로 또 나눕니다.

---

## 2) 가입할 때 정치색을 고르지 않음

- 처음엔 **본인이 진영을 고르지 않습니다.**  
- 대신 **글, 댓글, 좋아요, 싫어요** 등 활동을 분석해서 **한쪽으로 많이 기울었다**고 판단되면  
  **입당 추천서**가 가고, **본인 의사와 관계없이** 그 진영으로 **편입**됩니다.  
- 편입되면 그 진영의 **1단계 게시판**이 **열립니다.**

> 나중에 구현할 때: “분석”은 AI일 수도, 규칙표일 수도 있습니다. 지금은 **숫자·이름만** `config` 폴더에 적어 두었습니다.

---

## 3) 게시판 단계(4단계) — 누가 읽고, 누가 쓰나

**중도를 제외한** 아래 영토마다 **4단계** 게시판을 둡니다.

- 보수  
- 진보  
- 깐따삐아 · 좌  
- 깐따삐아 · 우  

**1단계**

- **읽기:** 양쪽 진영 + 일반인 **모두** 가능  
- **쓰기·댓글:** 정치색(점수)이 **어느 정도 채워져야** 가능  

**2~4단계**

- 점점 **조건을 강하게** (읽기도 제한할 수 있음 — 기획에 맞게 조정)

---

## 4) 중도에서 정치색이 차면

중도에서 글쓰기·좋아요·댓글(특히 **다른 성향 유저와의 싫어요/좋아요**도 반영)로 **점수가 차면**,  
그때 **해당 영토**로 활동 무대가 옮겨집니다. (위의 자동 편입과 연결)

---

## 5) 영토마다 인원 수 → 화면 모습

인원이 많아질수록 **마을이 발전한 것처럼** 보이게 하고 싶다는 뜻입니다.

**예시 (숫자는 `config/world-territories.js`에서 바꿀 수 있음)**

| 대략 인원 | 보여주는 느낌(예) |
|-----------|-------------------|
| 1 ~ 500 | 모닥불·부족 느낌 |
| … | 중간 단계 |
| 5,000 이상 | 길·마을·농경 사회 느낌 |

---

## 6) AI가 토론 거리를 던져 줌

- **일간 / 주간 / 데일리 / 일반** 같은 칸에  
- AI가 **토론 주제**를 올리고  
- 양 진영이 **대화할 거리**를 만들고 싶다는 기획입니다.
