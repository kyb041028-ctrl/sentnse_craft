# 나중에 Supabase에 만들 “표” 이름만 적어 둠

지금 있는 `profiles` / `identity_history` 는 **회원 명함·신분 기록**용입니다.  
아래는 **센텐스크래프트 기획**을 위해 **앞으로 추가할 수 있는** 표 이름입니다. (아직 SQL 파일은 안 만듦)

| 표 이름(예시) | 하는 일 (쉬운 말) |
|---------------|-------------------|
| `user_alignment_scores` | 유저마다 정치색 점수(보수 쪽 몇 점, 진보 쪽 몇 점…) |
| `alignment_events` | 글 올림, 댓글, 좋아요 등 **한 줄씩 기록** |
| `party_invites` | 입당 추천서가 언제 갔는지, 자동 편입됐는지 |
| `boards` | 게시판 한 칸 한 칸 (영토 + 1~4단계) |
| `posts` / `comments` | 글 / 댓글 본문 |
| `reactions` | 좋아요·싫어요 (누가 누구 글에) |
| `territory_stats` | 영토별 인원 수 — 화면 “발전 단계”에 쓰기 |
| `ai_topic_runs` | AI가 낸 주제가 언제·어느 칸에 올라갔는지 |
| `ads_preferences` | 광고 보기 / 유료로 광고 끄기 같은 수익 관련 |

**원칙:** 비슷한 일은 한 표에 몰지 말고, **나중에 고치기 쉽게** 잘게 나눕니다.
