# 영토 그래픽 — 어떻게 준비하면 좋은지 (쉬운 말)

## 한 줄 요약

**① 그림 스타일을 먼저 정하고** → **② 진영·단계마다 필요한 장면 목록을 적고** → **③ PNG/WebP 같은 이미지 파일로 저장** → **④ 웹·앱에서 그 파일만 불러오기** 순서입니다.

“로얄 킹덤 / 로얄 매치” 같은 느낌 = **둥근 형태, 파스텔, 굵은 외곽선(선명한 윤곽), 귀여운 캐릭터·건물** 쪽으로 **레퍼런스(참고 그림)** 몇 장을 모아 두면 됩니다.

---

## 그림으로 (준비 흐름)

```mermaid
flowchart LR
  A[스타일 정하기] --> B[필요한 그림 목록]
  B --> C[파일로 저장]
  C --> D[웹·앱에 연결]
```

---

## 1) 스타일 정하기 (가장 먼저)

| 할 일 | 쉬운 말 |
|--------|---------|
| **무드판** | “이런 느낌” 예시 이미지 5~10장 모아 두기 (색, 둥글기, 두꺼운 선) |
| **색 팔레트** | 메인 5~8색만 정해 두기 (보수/진보/중도/깐따삐아마다 살짝 다른 톤도 가능) |
| **선 규칙** | 외곽선 두께, 그림자 넣을지 말지 |

나중에 그림사람에게 맡기든, AI로 초안을 내든 **이 판만 있으면** 그림체가 안 흐트러져요.

---

## 2) 영토용으로 “무슨 그림”이 필요한지

지금 기획(인구 구간별 발전 단계, 진영 맵)과 맞추면 예시는 이런 식입니다.

- **메인 맵 한 장** — 보수 / 중도 / 진보 / 깐따삐아가 보이는 **한눈에 보는 지도**  
- **진영별 작은 아이콘** — 깃발, 성, 마을, 모닥불 같은 **작은 그림**  
- **단계별 배경 3~4장** — “원시 → 마을 → 도시” 같은 **발전 단계**마다 배경이 바뀌게  
- **(선택) 캐릭터** — 유저를 대표하는 **작은 캐릭터** 실루엣

처음엔 **맵 1장 + 진영 아이콘 4개 + 단계 배경 4장** 정도만 있어도 데모가 됩니다.

---

## 3) 파일로 저장할 때 (웹·앱 둘 다 잘 되게)

| 형식 | 언제 쓰나요 |
|------|-------------|
| **PNG / WebP** | 가장 무난. 배경·맵·버튼 그림. **WebP**는 용량이 작아서 앱에 좋아요. |
| **SVG** | 아이콘·깃발처럼 **크기를 크게 해도 안 깨지는** 단순 그림 |
| **JSON + 스프라이트** | 나중에 애니메이션(불 깜빡임 등) 넣을 때 (Spine, Lottie 등은 단계가 한 단계 올라갈 때) |

**해상도:**  
- 맵 전체: **가로 2배~3배 크기**로 그려 두고, 화면에서는 줄여서 쓰면 선이 예쁘게 보입니다.  
- 아이콘: **정사각형 256 또는 512** 픽셀 한 벌이면 웹·앱 둘 다 재사용하기 좋아요.

---

## 4) 이 프로젝트 폴더에 두는 위치 (제안)

나중에 헷갈리지 않게:

```
public/
  assets/
    territory/
      map-main.webp
      belt-conservative.webp
      ...
```

앱은 **나중에** 만들 때, 같은 파일을 복사하거나 **서버 URL로만** 불러와도 됩니다.

---

## 5) 그림은 누가 그리나요?

| 방법 | 장단점 (짧게) |
|------|----------------|
| **외주 일러스트** | 스타일 가장 통일됨. 비용·시간 있음 |
| **본인 + 간단 도구** (Procreate, 클립스튜dio, Aseprite) | 천천히라도 **스타일판**부터 |
| **AI로 초안** | 빠름. **상업 이용·저작권** 규칙만 꼭 확인 |

---

## 6) 코드와 연결 (나중에)

`config/world-territories.js` 에 있는 **발전 단계 이름**(예: `TRIBAL`, `TOWN`)과 **같은 이름**으로 이미지 파일을 맞춰 두면,  
프로그램에서 “지금 단계 = 이 파일”만 바꿔 끼우기 쉽습니다.

---

**정리:**  
먼저 **스타일판 + 필요한 그림 목록**만 잡고, **WebP/PNG로 `public/assets/territory/`** 에 모아 두면, 웹이든 앱이든 **같은 그림**을 쓸 수 있어요.
