영토 맵 편집 도구

서버가 떠 있는 주소 기준으로 아래 링크를 사용하세요. (로컬 예: http://localhost:3000) 영토는 한 장 PNG + 히트존 JSON으로 나뉩니다. 히트존 편집기에서 배경 미리보기를 끄면(localStorage 키 scTerritoryEditorShowIslandBg=0) 어두운 바탕만 보입니다.

  • 영토 레이아웃 안내
    단일 PNG 경로·히트존 편집기 링크 (박스 레이아웃 없음)
  • 마우스로 경계 찍기 — 히트존 편집기
    맵을 클릭해 꼭짓점을 순서대로 이어 폴리곤을 만듭니다. 점은 드래그로 이동. 저장: territories/territory-hit-zones.json
