영토 히트 경계 편집기

마우스로 경계 그리기
  1. 위에서 편집 구역(개척영토·수호영토·중앙광장·외계행성)을 고릅니다.
  2. 아래 맵에서 클릭할 때마다 꼭짓점이 생기고, 이전 점과 실선으로 이어집니다.
  3. 점이 세 개 이상이면 녹색 점선으로 닫힘 미리보기가 보입니다. 첫 점(큰 초록 테두리) 근처를 다시 클릭하면 닫힌 폴리곤으로 확정됩니다.
  4. 꼭짓점은 드래그로 옮길 수 있고, 마지막 점 취소로 되돌릴 수 있습니다.
  5. JSON은 편집할 때마다 아래 칸에 자동 반영됩니다. 확인 후 public/territories/territory-hit-zones.json에 저장하세요.

저장: 아래 JSON을 public/territories/territory-hit-zones.json 에 덮어쓰세요. 좌표는 맵 이미지(territory-zones-tribal-s1.png)가 contain으로 붙은 영역 기준 viewBox 0 0 1600 900 (letterbox 검은 여백 제외, 이미지 실렌더 영역 내부만) — 메인 SVG와 동일합니다.

통합 게시판·외계행성의 위치·크기는 territory-layout.json + 레이아웃 편집기입니다.
