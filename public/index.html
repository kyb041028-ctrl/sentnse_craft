센텐스크래프트

영토를 선택하면 해당 영토 전용 게시판으로 이동합니다. 기본 메인은 중도(CENTRIST) 게시판입니다.

영토 구역

픽셀 지도를 클릭하면 해당 영토 게시판으로 이동합니다. 하늘의 두 구체는 깐따삐아 좌·우, 아래 땅·바다는 보수·중도·진보입니다. 소속 인구(맥스 50만)에 따라 발전 단계 1~40과 가로 너비가 바뀌고, 시각 티어 1~10에 맞춰 픽셀 연출이 달라집니다. localStorage 키 sc_map_pop_v1(JSON). 예: __scTerritoryPop.set({ CONSERVATIVE: 800, CENTRIST: 120, PROGRESSIVE: 0, KANTAPBIYA_LEFT: 400 })

지금 이 화면이 보이면 정상입니다.

예전에 섞였던 “다른 실험 화면”은 빼 두었습니다. 앞으로 화면 파일은 public 폴더만 보면 됩니다.

  • 서버 실행: 터미널에서 npm start
  • 주소창: http://localhost:3000 (파일을 더블클릭해서 열면 안 됩니다)
  • 기획 정리 문서: 프로젝트 안 docs/PRODUCT_센텐스크래프트_기획.md

규칙 표(API로도 내려감): /api/public-config

데일리 이슈 · 오늘
