영토 구역

셀을 누르면 해당 영토 게시판으로 이동합니다. 중도 메인 랜드 위에 보수·진보 진영이 올라가 있고, 깐따삐아 좌·우는 하늘에 떠 있는 행성으로 표시됩니다. 보수·진보·깐따삐아(좌·우)는 소속 인원에 따라 1~4단계 일러스트가 바뀝니다(0~100 → 1, 101~200 → 2, 201~500 → 3, 501+ → 4). 메인 랜드 배경은 고정입니다. 베타 인원: localStorage 키 sc_belt_pop_v1(보수·진보), sc_kanta_pop_v1(깐따삐아). 예: __scTerritoryPop.set({ CONSERVATIVE: 150, KANTAPBIYA_LEFT: 220 })

지금 이 화면이 보이면 정상입니다.

예전에 섞였던 “다른 실험 화면”은 빼 두었습니다. 앞으로 화면 파일은 public 폴더만 보면 됩니다.

  • 서버 실행: 터미널에서 npm start
  • 주소창: http://localhost:3000 (파일을 더블클릭해서 열면 안 됩니다)
  • 기획 정리 문서: 프로젝트 안 docs/PRODUCT_센텐스크래프트_기획.md
