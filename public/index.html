센텐스크래프트

글로 싸우는 전쟁 — 정치·사회·금융 등 이슈를 게시판으로 다루는 공간 (준비 중)

영토 구역 (경계만 · 그림은 나중에)

가로 한 줄 = 메인 벨트(보수—중도—진보). 아래 보라 테두리 = 깐따삐아 행성 안 좌·우.

보수 진영 CONSERVATIVE · 게시판 4단계
중도(일반) 진영 CENTRIST · 정치색 연습장
진보 진영 PROGRESSIVE · 게시판 4단계
깐따삐아 (별도 행성)
깐따삐아 · 좌파 영토 KANTAPBIYA_LEFT · 4단계
깐따삐아 · 우파 영토 KANTAPBIYA_RIGHT · 4단계

지금 이 페이지가 보이면 정상입니다.

예전에 섞였던 “다른 실험 화면”은 빼 두었습니다. 앞으로 화면 파일은 public 폴더만 보면 됩니다.

  • 서버 실행: 터미널에서 npm start
  • 주소창: http://localhost:3000 (파일을 더블클릭해서 열면 안 됩니다)
  • 기획 정리 문서: 프로젝트 안 docs/PRODUCT_센텐스크래프트_기획.md

규칙 표(API로도 내려감): /api/public-config
